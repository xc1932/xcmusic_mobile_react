##  特性
### 技术特性
-  技术栈：React + React Router + Redux +AntD Mobile 全家桶
-  滚动特效：better-scroll
-  图片懒加载：react-lazy-load-image-component
-  视频播放：plyr
-  图标库：IconPark + AntDMobileIcons
### 功能特性
-  支持网易云账号登录（手机/邮箱登录）
-  支持每日推荐歌曲
-  支持搜索(歌单、专辑、艺人)
-  支持歌词显示
-  支持 MV 播放
-  支持歌曲下载

详细功能参考下方截图或在线体验

## 下载运行
### 下载
```js
git clone https://github.com/xc1932/xcmusic_mobile_react.git
```
### 安装依赖
```js
npm install
```
### 运行
```js
npm start
```
### 打包
```js
npm run build
```
